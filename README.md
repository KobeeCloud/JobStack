## Tech Stack

- **Frontend**: Next.js 15 (App Router), TypeScript, TailwindCSS
- **Backend**: Next.js API Routes, Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **Cache**: Upstash Redis
- **Scrapers**: Cloudflare Workers
- **AI**: Groq (Llama 3)
